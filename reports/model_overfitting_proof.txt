======================================================================
  TEST 4 — PERMUTATION TEST  (gold standard for genuine signal)
======================================================================

  The gold-standard test for overfitting / data leakage.
  If the model learned genuine patterns: shuffling the labels
  (while keeping features unchanged) should completely destroy performance.
  If performance survives label shuffling: the model is memorising artefacts.

  Method: shuffle y_val labels 200 times, compute F1 each time.
  p-value = fraction of shuffles where shuffled F1 ≥ real F1.

  N shuffles                : 200
  Real F1 on validation     : 0.9897
  Shuffled F1 — Mean        : 0.4431
  Shuffled F1 — Max         : 0.4575
  Shuffled F1 — Std         : 0.0055
  p-value                   : 0.0000  (p<0.05 = significant)
  Z-score (real vs shuffled): 99.2 standard deviations

  [✓ PASS]  p = 0.0000 < 0.05 → HIGHLY SIGNIFICANT. Model learned REAL patterns, not noise.
  [✓ PASS]  Real F1 (0.9897) > max shuffled F1 (0.4575) → IMPOSSIBLE TO ACHIEVE BY CHANCE
